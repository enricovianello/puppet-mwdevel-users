## Description

Add SDDS users and their public RSA keys.

## Setup

This module is available on puppet forge:

```
puppet module install cnafsd-sdds_users
```

## Usage

Use this module as follow:

```
include sdds_users
```

## Limitations

It works only on RedHat distributions.
